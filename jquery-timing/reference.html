.join([queueName],[callback])

The join method postpones the invocation of the following methods (tic) until the end of a jQuery effects queue is reached.

This requires jQuery 1.7 or above.

queueName
The optional name of the jQuery effects queue (fxq) to wait for. Defaults to default queue 'fx'.
callback
An optional callback function which is called after reaching the queue's end.
The arguments are the current iteration numbers of the surrounding repeat loops, starting from the innermost to the outermost loop.

.repeat(interval,[firstRunNow],[callback])
.repeat([interval],[callback])

The .repeat method loops all the following methods in the concatenated method chain (tic) up to an .until method, but concerning nested loops.

When given an interval internally winodws.setInterval() is used. Without an interval the loop iterations have no delay (except, of course, you put some .wait() in there).

interval
The interval in milliseconds (ms) to wait between each call of the following method chain (up to the first .until method). If skipped or if the value is zero then there is no delay between the iterations.
firstRunNow
A boolean whether to invoke the methods for the first time right now or after the first interval timeout. Defaults to false.
callback
An optional callback function to be run at the beginning of every iteration.
The arguments are the current iteration numbers of the surrounding repeat loops, starting from the innermost to the outermost loop.

$.repeat([threadName],interval,[firstRunNow],[callback])
$.repeat([threadName],[interval],[callback])

The static version of the repeat method can be used quite similar to the version above, but uses an optional timing thread name instead of a selection.

threadName
A timing thread name to be used for the static $.unrepeat(threadName).
interval
The interval in milliseconds (ms) to wait between each call of the following method chain (up to the first .until method). If skipped or if the value is zero then there is no delay between the iterations.
firstRunNow
A boolean whether to invoke the methods for the first time right now or after the first interval timeout. Defaults to false.
callback
An optional callback function to be run at the beginning of every iteration.
The arguments are the current iteration numbers of the surrounding repeat loops, starting from the innermost to the outermost loop.

.then(callback)

The then method just runs the given callback function at the time of invocation.

callback
A function which is called in the context of the jQuery selection object.
The arguments are the current iteration numbers of the surrounding repeat loops, starting from the innermost to the outermost loop.

$.then([threadName],callback)

The static version of the then method can be used quite similar to the version above, but uses an optional timing thread name instead of a selection.

threadName
A timing thread name to be used for the static $.unwait(threadName) or $.unrepeat(threadName).
callback
A function which is called in the context of the jQuery selection object.
The arguments are the current iteration numbers of the surrounding repeat loops, starting from the innermost to the outermost loop.

.unrepeat()

Interrupts ongoing .repeat operations on the current selection. That means that any currently repeated method chain for that selection will not start over again.

$.unrepeat([threadName])

The static version of the unrepeat method can be used quite similar to the version above, but uses an optional timing thread name instead of a selection.

threadName
A timing thread name from the static $.repeat(threadName,...).

.until()
.until(nrOfIterations)
.until(conditionCallback)

Defines that the last usage of .repeat before this call in the method chain (tic) only loops up to here. Any following command in the method chain (tic) is processed only once - after the last repetition of the loop.
The condition when to stop the repeat loop can be defined in three ways:

- no parameter -
Without a parameter the .until function checks whether the jQuery selection is empty or not. If the selection is non-empty then the loop is going on. If the selection is empty then the loop stops and starts processing all methods in the method chain after this call to .until.
nrOfIterations
If a numeric parameter is given then it defines the number of loop iterations to be run before leaving the loop.
Values less or equal to zero stop the loop after its first execution.
conditionCallback
If a callback function is given as argument to .until then this function is called after each loop iteration to determine whether to leave the loop or not.
If the return value is numeric than it is compared to the number of passed iterations. Otherwise it is treated as boolean.
The arguments are the current iteration numbers of the surrounding repeat loops, starting from the innermost to the outermost loop.
The function is called in the context (this) of the current jQuery selection object (which is the same context that the .until method is called itself).

.unwait()

Interrupts ongoing wait operations on the current selection. That means that the currently waiting commands will not be processed anymore.

$.unwait([threadName])

The static version of the unwait method can be used quite similar to the version above, but uses an optional timing thread name instead of a selection.

threadName
A timing thread name from the static $.wait(threadName,...).

.wait([timeout],[callback])

The wait method uses window.setTimeout() to delay the following methods in the concatenated method chain (tic).

timeout
The timeout in milliseconds (ms) to wait until the following methods are processed.
If skipped then 0 is used.
callback
An optional callback function to be run at the beginning of every iteration.
The arguments are the current iteration numbers of the surrounding repeat loops, starting from the innermost to the outermost loop.

$.wait([threadName],[timeout],[callback])

The static version of the wait method can be used quite similar to the version above, but uses an optional timing thread name instead of a selection.

threadName
A timing thread name to be used for the static $.unwait(threadName).
timeout
The timeout in milliseconds (ms) to wait until the following methods are processed.
If skipped then 0 is used.
callback
An optional callback function to be run at the beginning of every iteration.
The arguments are the current iteration numbers of the surrounding repeat loops, starting from the innermost to the outermost loop.
