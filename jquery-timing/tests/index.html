What the test suite is about

More than 500 different test cases checked against 11 different jQuery versions for both plain and minimized script — the result are >10k single test runs.

Compared to the source code (25KB) and the production script (2K minified, gzipped) we are super proud to have nearly 500KB of test code.

If you want to be sure that jquery-timing plugin works correct in your browser just start any number of tests.
