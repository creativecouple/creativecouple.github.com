--- layout: jquery-timing-api-signature title: waiting for a jQuery.Deferred subtitle: .wait(deferred ,callback) signature: .wait(deferred) name: 'wait' description: Wait for a deferred object to resolve (jQuery, jQuery.Deferred, tic, …). friends: - unwait - join ---
deferred – any object with .promise()
e.g. a jQuery selection, a jQuery.Deferred, or another tic.
callback – function(count…), optional
a method that is called once just before continuing the invocation chain.
All current iteration counts (from inner-most to outer-most loop) are passed as arguments.

Tip: You can also wait for the current end of the elements' effect queues via .join().
