--- layout: jquery-timing-api-signature title: use wait on a static thread group subtitle: $.wait(threadGroup, …) signature: $.wait() description: Do a wait in a static threadgroup. friends: - unwait ---
threadGroup – string, optional
a name of the thread group that this waiting should be assigned to.
Defaults to the empty string ''.
…, further parameters
can be used like in the non-static versions of .wait()

The static $.unwait() allows to break wait states defined this way.
