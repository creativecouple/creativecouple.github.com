--- layout: jquery-timing-api-signature title: breaking waits of a static thread group subtitle: $.unwait(group, continue) signature: $.unwait() name: 'unwait' description: Stop ongoing .wait()s in the named thread group. friends: - wait ---
group – string, optional
a name of the thread group whose waiting invocation chains should be left.
Defaults to the empty string ''.
continue – boolean, optional
whether to continue the interrupted chain for the selected elements or not
Defaults to false.
