--- layout: jquery-timing-api-signature title: waiting for a jQuery.Deferred in a thread group subtitle: $.wait(group, deferred ,callback) signature: $.wait(deferred) description: Wait for a deferred object to resolve (jQuery, jQuery.Deferred, tic, …) in a static thread group. friends: - unwait ---
group – string, optional
a name of the thread group that this waiting should be assigned to.
Defaults to the empty string ''.
deferred – any object with .promise()
e.g. a jQuery selection, a jQuery.Deferred object to wait for resolving.
callback – function(count…), optional
a method that is called once just before continuing the invocation chain.

The static $.unwait() allows to break wait states defined this way.
