--- layout: jquery-timing-api-signature title: static usage of repeat loop subtitle: $.repeat(threadGroup, …) signature: $.repeat() description: Begin a sequential loop belonging to a static threadgroup. friends: - until - unrepeat ---
threadGroup – string, optional
a name of the thread group that this loop should be assigned to.
Defaults to the empty string ''.
…, further parameters
can be used like in the non-static versions of .repeat()

The static $.unrepeat() allows to break repeat-loops started this way.
