--- layout: jquery-timing-usage title: How to attach an inline event handler using .on() signatures: - .on($) old: | $('.some').on('click', function(){ $(this).doSome().jQueryStuff(); }); new: | $('.some').on('click').doSome().jQueryStuff(); ---

The jQuery object is used as inline handler if the callback function is skipped or given as jQuery token $. Thus you can also write:

{% highlight javascript linenos %} $('.some').on('click',$).doSome().jQueryStuff(); {% endhighlight %}
