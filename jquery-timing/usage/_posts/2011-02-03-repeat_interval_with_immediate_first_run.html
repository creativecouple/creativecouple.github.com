--- layout: jquery-timing-usage title: How to repeat on interval with immediate first run signatures: - .repeat(interval) - .until(bool) old: | function myFunction(){ $('.some').doThis().overAndOver(); } myFunction(); setInterval(myFunction, interval); new: | $('.some').repeat(interval, true).doThis().overAndOver(); ---

The closing command .until() can be skipped for infinite loops. If you want to use it, write:

{% highlight javascript linenos %} $('.some').repeat(interval, true).doThis().until(false); {% endhighlight %}
