--- layout: jquery-timing-usage title: How to repeat the same actions on interval signatures: - .repeat(interval) - .until(bool) old: | setInterval(function(){ $('.some').doThis().overAndOver(); }, interval); new: | $('.some').repeat(interval).doThis().overAndOver(); ---

The closing command .until() can be skipped for infinite loops. If you want to use it, write:

{% highlight javascript linenos %} $('.some').repeat(interval).doThis().until(false); {% endhighlight %}
