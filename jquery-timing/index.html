timing — a jQuery plugin
introducing the Timed Invocation Chain (tic) as counterpart to the jQuery Effects Queue (fxq)

jquery-timing provides easy-to-use methods to define timeouts and intervals just in line with all your jQuery stuff.
It supplements the existing functionality from the jQuery effects queue (fxq) by introducing a new timed invocation chain technique (tic).

Follow the navigation for documentation, examples, references, and further information.
– Have fun!

Simplicity

If you like the simplicity of this very first example then I promise you that this plugin will be a good friend in all your future jQuery projects:

This is a sample sentence with some words.
var X=$$();
$('.example span').repeat(80,X).eq(X.mod(15)).toggleClass('bold');

Dependencies

You need to use jQuery on your page, any version since 1.2 and above.

Include in your page

All you need is to include the javascript file jquery-timing.min.js:

<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/<preferred-version>/jquery.min.js"></script>
<script type="text/javascript" src="http://creativecouple.github.com/jquery-timing/jquery-timing.min.js"></script>
