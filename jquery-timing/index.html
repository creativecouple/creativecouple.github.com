Simplicity

If you like the short syntax of these very first examples then we promise you that this plugin will be a good friend in all your jQuery projects:

{% highlight javascript %} $('.example span').repeat(1000) .each($).toggleClass('bold').wait(50); {% endhighlight %} ⇒

This is a sample sentence with some words.

{% highlight javascript %} $('button').on('click').next().css({left:'+=20px'}); {% endhighlight %} ⇒

Watch me moving!

Dependencies

You need to use jQuery on your page, any version since 1.2 and above.

If you want to combine timing with animations then you should use jQuery 1.4 or above. See our tests section for details.

Include in your page

All you need is to include the javascript file jquery-timing.min.js after jQuery:
