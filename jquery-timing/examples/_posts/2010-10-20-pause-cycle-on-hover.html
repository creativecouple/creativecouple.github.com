--- layout: jquery-timing-example title: pause animation cycle on mouseover signatures: - .animate($) - .wait(deferred) - .wait(callback) - .wait(event) - .each($) - .repeat() tags: - animations - each-loops - timeouts - deferreds - open loops ---

There is a animation cycle rotating three images. Once you move the mouse cursor over it, it pauses until the mouse is moved away.

A little helper function checks whether the mouse is over the image. If that's the case a deferred jQuery object waiting for the mouseleave event is returned. {% highlight javascript %} function noHover() { return this.is(':hover') ? this.wait('mouseleave') : this } {% endhighlight %}

The running loop can be written in a single jQuery line.

{% highlight javascript %} $('#images > img').hide().repeat().each($).fadeIn($) .wait(1000).wait(noHover).fadeOut($); {% endhighlight %} ⇒
